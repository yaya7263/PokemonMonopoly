How to build:

In QtCreator.

1. Open Project
2. Open PikaMonopoly.pro
3. Build with the latest version of QT.


How to Play:
1. Enter number of Players and number of Humans ( currently works for 2,2)
2. Press Start button
3. Roll Dice button
4. Player Move button
5. Decide on if you want to purchase (if so, press Purchase button)
6. Next Turn Button
7. Repeat from Step 3, in which it is the next player's turn
